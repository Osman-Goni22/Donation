Important FAQ

What is the Document Object Model (DOM)?

Every HTML file is converted as a tree called DOM. To manipulate any element of the tree we use DOM

How do you select an element from the DOM?

We can select an element using several ways. Among them using id, class, tagName, attributes etc are very common.

What is Event Delegation in the context of DOM ? and why it is useful?

Event Delegation is a technique in javascript that allows you to handle events at a higher level in the DOM rather than attaching event listener directly to individual elements.This approach takes advantages of event buggling phase.

How do you manipulate an element's attributes and styles using DOM?

We can manipulate an element's attributes by using setAttributes and get attributes methods and use style keyword to style.
